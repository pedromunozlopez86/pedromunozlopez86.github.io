# pedromunozlopez86.github.io

Sitio estático sencillo para la página de usuario de GitHub Pages de **Pedro Muñoz López**.

## Archivos principales

- `index.html`: estructura y contenido del sitio
- `styles.css`: estilos responsive del portafolio
- `README.md`: documentación básica del proyecto y publicación

## Cómo editar el contenido

1. Abre `index.html`.
2. Actualiza textos, enlaces o secciones según lo necesites.
3. Si quieres ajustar colores, espaciados o layout, edita `styles.css`.
4. Guarda los cambios y súbelos a la rama `main` para publicarlos en GitHub Pages.

## Cómo publicar en GitHub Pages

1. Ve a **Settings > Pages** en el repositorio.
2. En **Build and deployment**, selecciona **Deploy from a branch**.
3. En **Branch**, elige `main` y la carpeta `/ (root)`.
4. Guarda la configuración.

## URL final esperada

https://pedromunozlopez86.github.io/
